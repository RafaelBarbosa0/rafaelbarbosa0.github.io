Circuit Puzzle creator for Unity

An asset for the Unity engine that allows users to create and customize a fully functional Circuit Puzzle and integrate it into their game with no coding required.

Features:
- Create a grid with any desired number of rows and columns.
- Switch the pieces on the grid to design the puzzle to your liking.
- Pick between 5 different piece types and rotate them freely.
- Use events to choose what happens when the puzzle is completed.
- Easily assign inputs to control the puzzle.
- A bunch of options to customize functionality.
- Use your own custom models for each piece type.
